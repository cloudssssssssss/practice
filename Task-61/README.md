# Express Server with Middlewares (HW 61.1)

Розширення базового сервера Express системою мідлварів для логування, аутентифікації та вадідації.

## Нові можливості
Додано наступні мідлвари:
- **Logger:** Фіксує час, метод та URL кожного запиту в консолі сервера.
- **Auth Middleware:** Імітує перевірку авторизації для маршрутів `/users`.
- **Validator:** Перевіряє вхідні дані для POST-запитів.
- **Access Control:** Спеціальна перевірка прав для доступу до розділу статей.
- **Error Handler:** Глобальний відловлювач помилок для запобігання падінню сервера.

## Встановлення
1. `npm install`
2. `node src/app.js`

## 🛠 Опис обробки маршрутів
- **`/`**: Використовує тільки `logRequests`.
- **`/users`**: Вимагає заголовок `Authorization`. Без нього поверне 401.
- **`/articles`**: Проходить через `checkArticleAccess`, що логує перевірку прав у термінал.
