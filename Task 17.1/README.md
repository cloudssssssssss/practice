# Final Web Project - Responsive Landing Page

## Опис проєкту
Це фінальна робота, що демонструє навички сучасної верстки. Сайт повністю адаптивний та використовує семантичний HTML.

## Технології
- **HTML5**: Семантичні теги.
- **SCSS**: Використання змінних, вкладеності та міксинів.
- **JavaScript**: Реалізація мобільного меню (гамбургер).
- **Flexbox/Grid**: Для побудови сітки.

## Як запустити
1. Склонуйте репозиторій.
2. Відкрийте `index.html` у вашому браузері.
