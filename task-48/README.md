# HW 48.1: UI Libraries Integration (Ant Design)

Проект демонструє інтеграцію бібліотеки **Ant Design** у React додаток.

### Що зроблено:
- Встановлено та налаштовано `antd`.
- Використано компоненти: `Card`, `Button`, `Input`, `Typography`, `Space`.
- Додано іконки з пакета `@ant-design/icons`.
- Створено адаптивну картку з формою.

### Запуск:
`npm install`
`npm run dev`
