# HW 47.1: React Forms (Formik + Yup)

Проект реалізує форму реєстрації з валідацією в реальному часі.

### Використані бібліотеки:
- **Formik**: для керування станом полів та обробки сабміту.
- **Yup**: для створення схеми валідації (email, довжина пароля, обов'язкові поля).

### Як запустити:
1. `npm install`
2. `npm run dev`
