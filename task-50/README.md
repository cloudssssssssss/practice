# HW 50.1: Optimization with useMemo, useCallback, and React.memo

Цей проект демонструє методи оптимізації React-додатку для уникнення зайвих ререндерів.

### Використані техніки:
1. **React.memo**: Дочірній компонент `TodoItem` рендериться лише тоді, коли змінюються його пропси.
2. **useCallback**: Функція `toggleTodo` мемоізована, щоб не викликати ререндер дочірніх компонентів при зміні стану батьківського лічильника.
3. **useMemo**: Підрахунок виконаних завдань кешується і не перераховується при зміні незалежного стану (лічильника).

### Як перевірити:
Відкрийте консоль браузера. При натисканні на кнопку "Збільшити лічильник" ви не побачите логів "Render: ..." від списку справ.
