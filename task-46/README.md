# HW 46.1: Redux Thunk Async Actions
Рефакторинг проекту з використанням асинхронних запитів у Redux.

## Виконано:
- Використано `createAsyncThunk` для запитів до API.
- Обробка станів `pending`, `fulfilled`, `rejected`.
- Використано `axios` для HTTP-запитів.
- Дані завантажуються автоматично при старті додатка.
