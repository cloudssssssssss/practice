# Express + MongoDB Atlas Integration (HW 65.1)

Проект успішно інтегровано з хмарною базою даних MongoDB Atlas.

## Нові можливості
- **Mongoose Connection:** Стабільне підключення до Cluster0.
- **Data Models:** Створена схема `User` для структурованого зберігання даних.
- **Live Fetching:** Маршрут `/users-db` виконує запит до хмари та повертає реальні дані.

## Налаштування
1. Створіть файл `.env`.
2. Додайте `MONGO_URI` зі своїм паролем.
3. Запустіть сервер та насолоджуйтесь відсутністю бананів у базі даних!
