# Express Advanced Server: Static, Cookies & JWT (HW 63.1)

Просунута версія сервера з підтримкою безпеки, збереженням стану та обслуговуванням статичних файлів.

## Нові технології
- **JWT (JSON Web Token):** Реалізовано захист маршрутів. Доступ до `/users` можливий лише після отримання токена через `/login`.
- **Cookies:** Використовуються для зберігання JWT (з прапорцем `httpOnly`) та налаштувань теми оформлення користувача.
- **Static Files:** Налаштовано обслуговування папки `public` для віддачі `favicon.ico`.

## Функціональність
- **Auth System:** Маршрут `/login` генерує токен, який автоматично зберігається в куках браузера.
- **Theme Switcher:** Маршрут `/set-theme/:theme` дозволяє зберігати вподобання користувача між сесіями.
- **Unified Views:** Усі PUG та EJS шаблони містять посилання на фавіконку для цілісного вигляду сайту.

## Як користуватися
1. Запустіть сервер: `node src/app.js`.
2. Перейдіть на `/users` — ви отримаєте відмову (403).
3. Перейдіть на `/login`, щоб авторизуватися.
4. Тепер розділ `/users` стане доступним.
5. Спробуйте змінити тему через `/set-theme/dark`.
