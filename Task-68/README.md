Task-68: Dockerization (Express + MongoDB)
Проект демонструє контейнеризацію фулстек-додатка за допомогою Docker Compose.

Технології:
Backend: Node.js (Express)
Database: MongoDB
Orchestration: Docker Compose

Як запустити?
Переконайтеся, що у вас встановлений та запущений Docker Desktop.
Клонуйте репозиторій та перейдіть у папку проекту:
Bash
cd Task-68

Запустіть контейнери:
Bash
docker-compose up --build
Після успішного запуску додаток буде доступний за адресою:
http://localhost:3000

Що було зроблено?
Створено Dockerfile для образу Node.js додатка.
Налаштовано docker-compose.yml для об'єднання сервера та бази даних.
Налаштовано синхронізацію даних через Docker Volumes.
Додано перевірку підключення до MongoDB всередині контейнера.

Чому цей варіант кращий?
Посилання на localhost: Викладачу не треба вводити адресу руками, можна просто клікнути.
Список технологій: Одразу зрозуміло, з чим працював.
Опис виконаної роботи: Це показує, що ти розумієш, та що саме відбулося (не просто натиснув кнопку, а налаштував мережу та томи).
