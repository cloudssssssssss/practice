# HW 39.1. Компоненти React: Stateful проти Stateless

Цей проект демонструє різницю між компонентами зі станом (Stateful) та компонентами без стану (Stateless) у React.

## Опис проекту
- **StatefulComponent**: Керує внутрішнім станом за допомогою хука `useState`. Містить лічильник та кнопку для його збільшення.
- **StatelessComponent**: Отримує дані через `props` від батьківського компонента і просто відображає їх. Не має власного стану.
