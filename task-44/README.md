# HW 44.1: React Context API
Проект демонструє передачу глобальних станів (тема та дані користувача) через Context API.

### Особливості:
- Створено `AppProvider` для керування станом.
- Реалізовано 3 рівні вкладеності компонентів.
- Дані в `DeepChild` отримуються через `useContext`, минаючи `MiddleParent`.
- Додано можливість перемикання теми (Light/Dark).
