# 🎯 Task Manager - Redux Toolkit Course Project

Курсовий проект з React та Redux Toolkit.

### 🚀 Функціонал:
- Завантаження початкових даних з JSONPlaceholder API через `createAsyncThunk`.
- Додавання нових завдань локально.
- Перемикання статусу виконання.
- Видалення завдань.
- Повністю адаптивний інтерфейс.

### 🛠 Стек:
- **React 18** + **Vite**
- **Redux Toolkit** (Slices, Thunks)
- **Axios** (для запитів)
- **CSS3** (темна тема)

### 📦 Запуск:
1. `npm install`
2. `npm run dev` (розробка)
3. `npm run build` (збірка)
