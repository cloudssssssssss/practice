# Task-67: Cursor & Aggregation in MongoDB

Цей проект демонструє просунуті методи роботи з MongoDB Atlas через Mongoose.

## Нові можливості
1. **Cursors (`/users-cursor`)**: Реалізовано потокову обробку даних за допомогою `.cursor()`. Це дозволяє серверу працювати з великими колекціями, не перевантажуючи RAM.
2. **Aggregation Pipeline (`/stats`)**: Впроваджено складний аналітичний запит, який групує користувачів за статусом та обчислює середній вік, кількість та мінімальний вік у кожній групі прямо на боці БД.

## Як тестувати
1. Запустіть `node src/app.js`.
2. Перейдіть на `/seed` для створення тестових даних.
3. Перейдіть на `/stats`, щоб побачити результат агрегації.
4. Перейдіть на `/users-cursor` та спостерігайте за логами в консолі.
