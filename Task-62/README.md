# Express Multi-Engine Server (HW 60.1 - 62.1)

Комплексний навчальний проект, що поєднує архітектуру MVC, систему мідлварів та підтримку декількох шаблонізаторів.

## Особливості
- **Multi-Engine Rendering:** Використання PUG для розділу користувачів та EJS для розділу статей.
- **Middleware Layers:** Реалізовано логування запитів, імітацію аутентифікації та валідацію даних.
- **MVC Pattern:** Чіткий розподіл на моделі/контролери, маршрути та представлення (views).
- **Custom Styling:** Кожна сторінка має власне оформлення через вбудовані CSS стилі.

## Технологічний стек
- Node.js & Express.js
- PUG (Template Engine)
- EJS (Template Engine)

## Запуск проекту
1. Встановіть залежності: `npm install`
2. Запустіть сервер: `node src/app.js`
3. Відкрийте: `http://localhost:3000`

## Маршрути
- `GET /` - Головна сторінка зі списком розділів.
- `GET /users` - Список користувачів (PUG).
- `GET /articles` - Список статей (EJS).
- `GET /users/:id` & `GET /articles/:id` - Детальні сторінки.
