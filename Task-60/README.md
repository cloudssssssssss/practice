# Express RESTful API Server (HW 60.1)

Це навчальний проект, що демонструє створення модульного веб-сервера на базі Node.js та фреймворку Express.js. Архітектура проекту побудована за паттерном **MVC** (Model-View-Controller).

## Технології
- **Node.js**
- **Express.js**
- **ESM (ECMAScript Modules)** — використання `import/export`.

## Структура проекту
Проект має модульну структуру для полегшення підтримки та масштабування:
- `src/app.js` — точка входу, конфігурація сервера.
- `src/routes/` — опис маршрутів (ендпоінтів).
- `src/controllers/` — логіка обробки запитів.

## Встановлення та запуск

1. Склонуйте репозиторій.
2. Переконайтеся, що у вас встановлений Node.js.
3. Встановіть залежності:
   ```bash
   npm install
4. Запустіть сервер командою node src/app.js
Сервер почне прослуховування на http://localhost:3000.

Опис маршрутів (Endpoints)
Головний маршрут
GET / — повертає текстове повідомлення "Get root route".

Користувачі (Users)
GET /users — Отримати список усіх користувачів.

POST /users — Створити нового користувача.

GET /users/:userId — Отримати дані користувача за його ID.

PUT /users/:userId — Оновити дані користувача за ID.

DELETE /users/:userId — Видалити користувача за ID.

Статті (Articles)
GET /articles — Отримати список усіх статей.

POST /articles — Опублікувати нову статтю.

GET /articles/:articleId — Отримати статтю за її ID.

PUT /articles/:articleId — Редагувати статтю за ID.

DELETE /articles/:articleId — Видалити статтю за ID.

Усі відповіді сервера повертаються у текстовому форматі (text/plain), що відповідає технічному завданню для спрощення відладки та інтеграції.
