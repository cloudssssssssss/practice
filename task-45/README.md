# HW 45.1: Redux Toolkit Integration

Цей проект демонструє міграцію з локального стану на глобальне керування за допомогою Redux Toolkit.

## Використані технології:
- **Redux Toolkit**: для створення слайсів та налаштування стору.
- **React-Redux**: `useSelector` для отримання даних та `useDispatch` для виклику дій.

## Що реалізовано:
- Глобальний стейт лічильника.
- Можливість інкременту, декременту та скидання через Redux Actions.
- Повна інтеграція `Provider` на рівні всього додатку.
