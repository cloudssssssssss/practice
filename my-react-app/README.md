# Task 38.1: React + Vite Project

Цей проект — моя перша робота на React. Створений за допомогою Vite.

## Виконані завдання:
- Ініціалізація проекту через Vite.
- Створення функціональних компонентів `Button` та `Input`.
- Використання `props` та обробників подій (`onClick`, `onChange`).
- Керування станом за допомогою хука `useState`.

## Як запустити локально:
1. `npm install`
2. `npm run dev`
