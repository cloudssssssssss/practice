# MongoDB CRUD Operations & Express (HW 66.1)

Цей проект демонструє розширену роботу з MongoDB Atlas, включаючи всі основні операції маніпуляції даними (Create, Read, Update, Delete).

## Технологічний стек
- **Node.js & Express.js**: Серверна частина.
- **MongoDB Atlas**: Хмарна база даних.
- **Mongoose**: Бібліотека для моделювання даних.

## Опис маршрутів та методів

| Маршрут | Метод БД | Опис функціональності |
|:---|:---|:---|
| `/add-one` | `create` (insertOne) | Створює одного тестового користувача "Solo Hero". |
| `/add-many` | `insertMany` | Масово додає масив з двох об'єктів у колекцію. |
| `/users-names` | `find` + **Projection** | Повертає список імен, приховуючи `_id` та `email`. |
| `/update-one` | `updateOne` | Знаходить користувача за email та змінює його ім'я. |
| `/update-many` | `updateMany` | Оновлює статус `verified` для всіх активних записів. |
| `/replace-one` | `replaceOne` | Повністю замінює документ новим об'єктом. |
| `/delete-one` | `deleteOne` | Видаляє конкретний запис за фільтром. |
| `/delete-all` | `deleteMany` | Повністю очищує колекцію `users`. |

## 🚀 Як протестувати
1. Переконайтеся, що файл `.env` містить коректний `MONGO_URI`.
2. Запустіть сервер: `node src/app.js`.
3. Перейдіть на головну сторінку `http://localhost:3000`.
4. Використовуйте інтерактивне меню для виклику CRUD-операцій.

## Приклад відповіді (Projection)
Маршрут `/users-names` повертає чистий масив об'єктів:
`[{"name": "First Dev"}, {"name": "Second Dev"}]`
