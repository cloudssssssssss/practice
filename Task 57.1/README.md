# Pure Node.js HTTP Server (HW 57.1)

Базовий HTTP-сервер, написаний на чистому Node.js без використання сторонніх фреймворків (типу Express). Проєкт демонструє роботу з нативним модулем `http`, маршрутизацію, обробку POST-запитів та базові заходи безпеки.

## Технології
- **Node.js** (нативні модулі: `http`, `querystring`)
- **JavaScript ES6+**

## Запуск проекту

1. Переконайтеся, що у вас встановлено Node.js.
2. Склонуйте репозиторій або скопіюйте файл `server.js`.
3. Запустіть сервер командою:
   ```bash
   node server.js
