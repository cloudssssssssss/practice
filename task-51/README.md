# HW 51.1: Integration of Specialized Libraries

Проект демонструє використання трьох популярних бібліотек для React.

### Реалізовано:
1. **React Icons**: Використано іконки Font Awesome для візуалізації станів.
2. **React Toastify**: Система інтерактивних сповіщень (Success, Warning, Info).
3. **React Idle Timer**: Детектор бездіяльності користувача (спрацьовує через 5 секунд).

### Запуск:
`npm install`
`npm run dev`
