# HW 40.1. React: Controlled, Uncontrolled & API

Цей проект демонструє роботу з формами та асинхронними запитами в React.

## Що реалізовано:
- **ControlledForm**: робота з інпутом через `useState`.
- **UncontrolledForm**: доступ до даних через `useRef`.
- **DataFetcher**: отримання списку користувачів з JSONPlaceholder через `useEffect`.

## Запуск
1. `npm install`
2. `npm run dev`
